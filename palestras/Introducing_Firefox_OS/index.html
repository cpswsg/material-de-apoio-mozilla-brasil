Uma plataforma Web

A Mozilla criou uma plataforma livre e inteira em HTML5 + JS.
Isso mesmo. É incrível!

  1. Todo App feito para Firefox OS é compatível com Android.

  2. Qualquer pessoa que saiba HTML pode ter um App.

  3. Todos os apps são feitos como se fossem um simples site.

  4. A renderização do Javascript para games é altíssima.

  5. Através do Javascript você pode acessar o hardware, contatos, agenda e etc.

  6. Todo lucro gerado com o seu App pode ser só seu.

  7. Você pode ter a sua Firefox Marketplace.

Uma loja livre para você:

  1. Ter qualidade e segurança na hora de baixar/comprar App's.

  2. Distribuir seus App's de maneira simples e rápida.

  3. Usufruir da estrutura e qualidade de aprovação da Mozilla.

  4. Gerar renda com App's e Games.

  5. Baixar o código(é open source!) e fazer a sua própria Loja.

  6. Gerar sua própria renda, ou não.

  7. Utilizar os App's do FFOS no seu Android, PC, Mac ou Linux. =D

Como funciona o FFOS

A Mozilla fatiou o OS em 3 partes, que são:

  1. GONK


    Kernel Linux, Firmware, Drivers

  2. GECKO


    Motor Mozilla, Browser runtime

  3. GAIA/HAIDA


    Interface do usuário, App's, tudo em HTML5
